Extract Information with Less Code

It's a newborn project, but this is the kind of interaction we envision for you to interact with your data. You can also check out some working examples, or fork a template project to get started.

{-# LANGUAGE OverloadedStrings, NoImplicitPrelude #-}
import ClassyPrelude
import NLP.Albemarle

main = do
  my_stuff <- lines <$> readFile "mystuff.txt"
  let phrase_model = Phrases.find $ Tokens.wordTokenize <$> my_stuff
  let topic_model = LSI.generate $ Phrase.use phrase_model <$> my_stuff
  print topic_model
